# 실행 가이드

## 1. 프로젝트 설치
```bash
$ git clone https://github.com/gychoi7/backend-onboard-task.git
$ cd backend-onboard-task
$ go mod tidy
```

## 2. 환경 설정
```bash
$ docker-compose up -d
```

## 3. 데이터베이스 마이그레이션 및 서버 실행
```bash
$ go run main.go
```
